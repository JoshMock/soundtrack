Soundtrack

  1. Turn down your speakers (just to be safe)
  2. Allow access to your webcam
  3. Listen to wacky sounds that change based on the brightness, color, etc. of your webcam's output.

Soundtrack is synthesized audio generated with pure JavaScript using your webcam. Powered by WebRTC and the Web Audio API. This should work in any version of any browser that supports the Web Audio API and supports WebRTC.

Check the Github repo for more info!
